Short Codes

High-Volume SMS and MMS

Send and receive 100 messages per second from a single number with the Twilio Messaging API.

Twilio Short Codes

Powerful features in an elegant, straightforward API.

Features

100 Messages per Second

Exchange SMS and MMS in bulk from the same number for one-to-one and one-to-many use cases at 100 messages per second.

Vanity & Random Codes

Opt for recognizably-branded, or lower-cost carrier-assigned Short Codes. Twilio manages the 8-12 week carrier approval process for you.

Bi-Directional Unicode

Whether it's Kanji or Emoji, incoming and outgoing messages are formatted utilizing fixed and variable-length character encoding.

Picture Messages

MMS over short codes in the US

Message Queueing

Queueing of concurrent message requests

Concatenation

Assemble up to 10 messages into one

Global Coverage

Available in the US, Canada, and the UK

Delivery Intelligence

Build better application logic with delivery status

Two-Way API

Respond with TwiML and send with REST API
