Radar

Radar is an integrated solution for automatic fraud protection. If you need help after reading this, get in touch or chat live with other developers in #stripe on freenode.

Radar availability

We’re currently rolling out Radar’s features to all accounts over the next few weeks.

Radar is a suite of modern tools to help you fight fraud. It’s fully integrated into your payments flow and you can start using it without any additional development time. Radar offers the following key features:

  • Machine learning-based risk evaluations on all your card payments
  • Review unusual payments and take action through the Dashboard
  • Rules to automate custom business logic
  • Insights on the latest state of your fraud prevention, with metrics on automated Stripe actions, custom rules and review

Getting started

  1. Learn more about the risk evaluations that appear in the Dashboard and API
  2. Make sure your website or app uses Stripe.js, Checkout, or our mobile SDK integrations for iOS and Android. Radar is only available when using one of these methods to accept payment information.
  3. Learn best practices for reviewing payments
