Data Migration PGP Key

Encrypt sensitive data for PCI-compliant migration using this PGP key. If you have any questions, or encounter any issues, please contact us at security@stripe.com.

General PGP key

Our PGP import key is for migrating sensitive card information to Stripe. Use our general PGP key for any other purpose that requires encrypted communication.

To migrate sensitive data to Stripe, such as credit card information, it needs to be encrypted using this PGP key. After importing our key, you can encrypt files to send by running gpg --encrypt --recipient 61E33984 FILENAME. This creates FILENAME.gpg.
