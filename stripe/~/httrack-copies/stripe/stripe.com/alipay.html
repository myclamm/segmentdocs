Stripe + Alipay

Checkout now supports Alipay, China's #1 payment method. Alipay gives you instant access to a vast market. Once you enable Alipay, Checkout will automatically detect Chinese buyers and enable them to pay on web and mobile using instant SMS authentication.

Checkout anywhere

Stripe’s goal is to enable buyers everywhere. Checkout is a drop-in solution that eliminates the need for custom code to support different instruments and devices. Whatever the payment method, Stripe’s unified APIs work unchanged.
